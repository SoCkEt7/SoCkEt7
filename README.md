### 🚀 Main Project: Livediff
**Live Terminal Diff Watcher.** 

Livediff est un outil de productivité en Rust (Ratatui) qui permet de surveiller les changements de fichiers en temps réel. Idéal pour valider des générateurs de code, des refactors massifs ou des migrations sans attendre la fin du processus.

[**Explore Livediff**](https://github.com/SoCkEt7/Livediff) • `cargo install livediff`
